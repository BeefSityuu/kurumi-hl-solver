# Kurumi HL Solver  
  
Fabric 1.21.4 用 MOD

某鯖の "High and Low" の選択を分析し、判断を容易にするMODです。

クライアント側で完結する MOD となっています。
  
## 主な機能  
  
- High/Low それぞれの期待値及び抽選確率の計算
  
## 動作環境  
  
- Minecraft 1.21.4  
- Fabric Loader 0.16.14+  
- Fabric API  
- Fabric Language Kotlin 1.13.3+kotlin.2.1.21+
